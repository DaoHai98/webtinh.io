nội dung thẻ div 
      Thuộc tính relative khi sử dụng các thuộc tính đi kèm như top , left , right , bottom
    thì thẻ html đó sẽ di chuyển dựa trên cái vị trí ban đầu của nó
    giả sử
    ban đầu nó có toạ độ : 15*15 ( top: 15px ; left: 15px )
    nhưng khi sử dụng các thuộc tính css position:relative và top : 30px , left 30px )
    => toạ độ mới của nó sẽ lấy toạ độ cũ ban đầu + với các thuộc css  top , left , right , bottom
    để tạo ra toạ độ mới => top ( 45px ) left ( 45px )
